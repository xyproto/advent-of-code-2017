# Advent of Code 2017

Solutions in C++17

## Disclaimer

I am a beginner at C++17 and "modern" C++ in general.

This is a learning experience.

## Requirements

* g++ (GCC) 7 or later (or clang for `make clang`)
* [snakemake](https://github.com/xyproto/snakemake)

## Building

    snakemake

## Building and running tests

    snakemake test

## Cleaning

    snakemake clean
