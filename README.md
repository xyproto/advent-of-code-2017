# Advent of Code 2017

Solutions for Advent of Code 2017 in C++17.

The solution with the highest number is work in progress.

## Requirements

* g++ (GCC) 7 or later (or clang for `make clang`)
* [sakemake](https://github.com/xyproto/sakemake)

## Build

    sm

## Build and run tests

    sm test

## Clean

    sm clean
