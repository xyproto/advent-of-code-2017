# Advent of Code 2017

Solutions in C++17

## Disclaimer

I am a beginner at C++17 and "modern" C++ in general.

This is a learning experience for me.

## Requirements

* g++ (GCC) 7 or later (or clang for `make clang`)
* scons
* make

## Building

    make

## Building and running tests

    make test

## Cleaning

    make clean
